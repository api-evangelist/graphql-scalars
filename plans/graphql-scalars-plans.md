# GraphQL Scalars Plans

GraphQL Scalars is a fully open-source library published under the MIT license. There are no commercial tiers or paid plans for the library itself — all scalar types and functionality are freely available to any developer or organization.

## Open Source (Free)

- Full access to 80+ custom GraphQL scalar types
- DateTime, BigInt, EmailAddress, URL, UUID, IBAN, IPv4, JSONObject, and more
- Compatible with any GraphQL server (Apollo Server, GraphQL Yoga, Mercurius, etc.)
- TypeScript and JavaScript support
- MIT license — use in any project, commercial or non-commercial
- Community support via GitHub Issues and Discord

## Related Paid Services (The Guild — GraphQL Hive)
